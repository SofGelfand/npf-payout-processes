# npf-payout-processes
Process models for a major Russian non-state pension fund: payout applications and refusals
# Моделирование процессов крупного российского НПФ

## Описание репозитория
В этом репозитории представлены артефакты системного анализа, разработанные для **крупного российского негосударственного пенсионного фонда (НПФ)**.

Артефакты описывают ключевые процессы обработки заявлений клиентов:
- **ЗЕВ** — заявление на единовременную выплату средств пенсионных накоплений
- **ЗЕВО** — заявление об отказе от ранее поданного заявления на выплату
- **ОПЕВ** — итоговое решение по выплате

Все схемы отражают реальную бизнес-логику, включая проверки условий, взаимодействие с государственными органами (СФР) и жизненный цикл заявлений.
